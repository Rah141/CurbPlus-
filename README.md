# Liva Policy Checker (Streamlit)

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy to Streamlit Community Cloud

1. Push this folder to a GitHub repo.
2. Go to https://share.streamlit.io/
3. → New app.
4. Select your repo, branch (e.g., main), and Main file path = app.py.
5. Deploy.

### Notes
- Add secrets (if any) via the app's Settings → Secrets on Streamlit Cloud.
- If you change dependencies, update requirements.txt and push again.
